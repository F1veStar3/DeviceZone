JavaScript, JQuery, CSS, HTML, NPM, tailwindcss, flowbite

## Back-end
Python, Django, Celery, Flower, Allauth, Debug Toolbar, Django Compressor, Pytest

## DataBases
PostgreSQL,Redis

## Additional
Docker/Docker Compose, Git Actions, Stripe
